Privacy Policy

This privacy policy describes how the Nascenia Timetracker Chrome Extension ("we", "our", or "the extension") collects, uses, and handles your information.

Information Collection

The extension collects the following information:

  • Count of employees present and checked out from the Nascenia Timetracker system
  • Basic page content from the Nascenia Timetracker website to display checkout statistics

Use of Information

The information collected is used solely for:

  • Displaying real-time checkout statistics in the extension popup
  • Calculating the progress of daily checkouts

Data Storage

This extension:

  • Does not store any personal information
  • Does not maintain any persistent database
  • Only processes data temporarily in memory while the popup is open

Data Sharing

We do not:

  • Share any data with third parties
  • Send any data to external servers
  • Use the data for any purpose other than displaying statistics

Security

The extension:

  • Only accesses the Nascenia Timetracker website
  • Uses secure HTTPS connections
  • Operates entirely within your browser

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the "Last Updated" date below.

Contact

If you have any questions about this privacy policy, please contact the Nascenia IT team.
